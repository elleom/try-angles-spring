####Technologies used:
  * Java 11
  * JS

###Intructions:
_Run containerized_

`docker build -t springio/triangles-spring-boot-docker .`
`docker run -p 8080:8080 springio/triangles-spring-boot-docker`

_enter URL in your browser_

`localhost:8080`
